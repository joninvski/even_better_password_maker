Instructions
============

Note: Do not forget to set ANDROID_HOME variable.

To compile
----------

    ./gradlew compile

To deploy to phone
------------------

    ./gradlew installDebug

Test
----
    ./gradlew connectedInstrumentTest

Note
----

Do not forget to set and export the ANDROID_HOME and JAVA_HOME variables
